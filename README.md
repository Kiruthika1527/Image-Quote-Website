# Image Quote Website

A simple and attractive web page built using **HTML** and **CSS** that displays an inspirational quote over an image.

## Features
- Responsive image display
- Inspirational quote overlay
- Centered text using CSS
- Clean and simple user interface
- Beginner-friendly project

## Technologies Used
- HTML5
- CSS3
